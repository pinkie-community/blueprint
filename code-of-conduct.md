# Code of Conduct: 行動規範

## 1. はじめに

pinkie-community へようこそ！このドキュメントは、私たちのコミュニティに参加するすべての人が、お互いに敬意を払い、安全かつ快適に活動するための「行動規範」です。

私たちの理念である our-compass.md の精神を実現するため、ここに集うすべてのメンバーがこの規範を守ることを期待します。

私たちの目標は、年齢、性別、性的指向、障害、外見、人種、宗教、技術レベル、経験に関わらず、誰もが歓迎され、安心して貢献できるコミュニティを育むことです。

## 2. 私たちの誓い

私たちは、オープンで友好的、そして協力的な環境を維持するために、以下の行動を奨励します。

- **敬意と思いやりのあるコミュニケーションを心がける**: 相手の立場や意見を尊重し、丁寧な言葉遣いをしましょう。

- **建設的な意見を歓迎する**: 異なる意見は、コミュニティが成長するための貴重な財産です。批判ではなく、改善に繋がる建設的なフィードバックを歓迎します。

- **助け合いの精神を持つ**: 誰かが困っていたら、積極的に手を差し伸べましょう。質問は歓迎され、知識の共有は推奨されます。

- **異なる視点と経験に感謝する**: 私たちは、多様なバックグラウンドを持つ人々が集まることで豊かになります。自分とは違う考え方や経験から学ぶ姿勢を大切にします。

## 3. 受け入れられない行動

以下の行為は、コミュニティの価値観に反するため、一切許容されません。

### あらゆる形態のハラスメント:
- 性別、性的指向、人種、宗教、年齢、外見など、他者を不快にさせる個人的な属性に関する言動。
- 歓迎されない性的な言動や画像の共有。
- 脅迫、ストーキング、執拗な嫌がらせ。

### 侮辱的、差別的な言動:
- 他者を貶める、見下すような発言やジョーク。
- 暴力的、差別的な言葉や画像の投稿。

### プライバシーの侵害:
- 他者の個人情報（住所、メールアドレス、電話番号など）を、本人の明確な許可なく公開する行為。

### その他の破壊的行為:
- 意図的に議論を妨害する行為。
- その他、コミュニティの健全で協力的な雰囲気を損なうと判断されるあらゆる行為。

## 4. 違反を発見した場合

もし、あなた自身がハラスメントを受けたと感じたり、他の誰かがハラスメントを受けているのを目撃した場合は、速やかにコミュニティ運営者にご連絡ください。

**連絡先**: [ここに連絡先のメールアドレスやフォームへのリンクを記載]

すべての報告は、秘密厳守で慎重に取り扱われます。報告者の安全を最優先に対応いたします。

## 5. 違反した場合の対応

この行動規範への違反が確認された場合、コミュニティ運営者は、その内容と状況を鑑みて、適切かつ公正であると判断した対応を取ります。

対応には、以下のようなものが含まれます。

- 非公式な注意・警告
- 一時的なコミュニティへの参加停止
- 永久的なコミュニティからの追放

## 6. この規範の精神

この行動規範は、単なる禁止事項のリストではありません。これは、私たちが共に育んでいきたいと願う、コミュニティ文化の基礎となるものです。

ここに集う一人ひとりが、この規範の精神を理解し、実践することで、pinkie-community は、誰もにとって価値ある場所になると信じています。
